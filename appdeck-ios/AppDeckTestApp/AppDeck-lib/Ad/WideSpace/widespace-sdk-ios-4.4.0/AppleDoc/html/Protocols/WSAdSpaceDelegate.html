WSAdSpaceDelegate Protocol Reference

Conforms to NSObject
Declared in WSAdSpace.h

Overview

The delegate of a WSAdSpace object must adopt the WSAdSpaceDelegate protocol. Optional methods of the protocol allow the delegate to receive notifications about what is happening with the ads and the WSAdSpace.

All the methods of the WSAdSpaceDelegate protocol are optional.

Instance Methods

adSpaceShouldPerformAdEndAction:

Implement this method to instruct a finished ad (no more interactions) should do. You can tell it to stay on screen as a small collapsed ad or have it close completly and disappear. Ads that have collapsed are not allowed to expand again or make any interactions with the user. The default behaviour if not implementing this method is WSAdEndActionClose.

- (WSAdEndAction)adSpaceShouldPerformAdEndAction:(WSAdSpace *)adSpace

Parameters

adSpace

The AdSpace that has an ad that ended and wants to perform an action.

Return Value

The action the ad should perform.

AdSpace failed and is reporting an error, if this happens you should check what error it is and try to handle it.
